# Visão do Projeto — RetailFlow Pro

## Nome do Projeto

RetailFlow Pro

## Descrição

RetailFlow Pro é uma aplicação ERP/POS web para varejo, desenvolvida com Next.js, React.js, TypeScript, PostgreSQL, Prisma e integração real com Mercado Pago.

O projeto será construído como uma aplicação de portfólio sênior, demonstrando domínio de arquitetura full-stack moderna, design system, API modular, banco relacional, operação de PDV, fluxo financeiro, webhooks, testes automatizados, CI/CD e deploy.

## Objetivo Principal

Criar um sistema profissional que simule e implemente boa parte da operação real de uma loja de varejo, incluindo:

- painel administrativo;
- cadastro de produtos;
- cadastro de clientes;
- controle de estoque;
- abertura e fechamento de caixa;
- vendas em PDV;
- vendas offline-first;
- contas a receber;
- integração com Mercado Pago;
- webhooks de pagamento;
- dashboard gerencial;
- auditoria;
- documentação técnica.

## Objetivo de Portfólio

O RetailFlow Pro deve demonstrar senioridade técnica por meio de:

- arquitetura modular;
- monorepo organizado;
- código limpo e tipado;
- separação clara entre domínio, aplicação e infraestrutura;
- uso correto de PostgreSQL e Prisma;
- preocupação com segurança;
- documentação de decisões técnicas;
- testes automatizados;
- CI/CD;
- deploy funcional;
- integração real com serviço externo.

## Público-Alvo do Projeto

Este projeto não será construído inicialmente para venda comercial, mas sim como um projeto de demonstração profissional para:

- recrutadores técnicos;
- empresas de tecnologia;
- entrevistas de emprego;
- avaliação de senioridade;
- portfólio no GitHub;
- demonstrações práticas de arquitetura full-stack.

## Resultado Esperado

Ao final do projeto, o sistema deverá estar acessível publicamente, documentado, com dados de demonstração e fluxos completos funcionando.
