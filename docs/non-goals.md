# Fora do Escopo — RetailFlow Pro

Este documento define o que o RetailFlow Pro não pretende entregar nesta versão.

## Fora do Escopo da Primeira Versão

O sistema não irá implementar:

- emissão real de NFC-e na SEFAZ;
- SPED fiscal completo;
- TEF presencial real;
- integração com pinpad físico;
- conciliação bancária completa;
- contabilidade completa;
- tributação brasileira completa;
- cálculo fiscal real por UF;
- multiempresa avançado;
- marketplace com split de pagamento;
- antifraude próprio;
- certificação PCI DSS;
- alta disponibilidade enterprise;
- cluster Kubernetes;
- replicação avançada de banco;
- emissão real de boletos bancários fora do Mercado Pago;
- integração com balanças comerciais;
- integração com leitores físicos via driver nativo.

## Motivo dos Cortes

Esses itens foram removidos do escopo inicial porque aumentariam muito a complexidade técnica, jurídica, fiscal ou operacional do projeto.

O foco desta versão é demonstrar senioridade full-stack, arquitetura, integração de pagamento, operação de PDV, estoque, financeiro e boas práticas modernas de desenvolvimento.

## Possível Roadmap Futuro

Alguns itens podem ser adicionados futuramente, como:

- emissão fiscal real;
- conciliação bancária;
- TEF;
- multiempresa avançado;
- integração com marketplaces;
- integração com ERP externo;
- relatórios fiscais avançados;
- app mobile.
